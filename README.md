# SentinelAI

## Autonomous Multi-Agent Security Operations Center (AI-SOC)

SentinelAI is an AI-powered Security Operations Center (AI-SOC) designed to automate the initial stages of cybersecurity incident investigation using Large Language Models (LLMs), autonomous AI agents, and modern web technologies.

The platform assists security analysts by analyzing uploaded security logs, detecting suspicious activities, assessing incident severity, generating investigation reports, and providing AI-powered explanations and recommendations.

---

## Objectives

- Automate security log analysis
- Assist SOC analysts using AI
- Detect suspicious activities and attack patterns
- Generate AI-assisted investigation reports
- Demonstrate enterprise-grade AI and cybersecurity architecture
